# Random Number Generator Algorithm (LCG)

## Description
This repository contains an implementation and analysis of a Linear Congruential
Generator (LCG) random number algorithm.

## Contents
- random_generator.py : RNG implementation
- quantization.py     : Quantization table generation
- analysis_report.md  : Algorithm analysis

## How to Run
```bash
python random_generator.py
python quantization.py
